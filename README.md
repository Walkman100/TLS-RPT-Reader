# TLS-RPT-Reader [![Build status](https://ci.appveyor.com/api/projects/status/3afnqvsncce4g8vt)](https://ci.appveyor.com/project/Walkman100/TLS-RPT-Reader)
SMTP TLS Report reader for MTA-STS policies

- EasyDMARC info page: https://easydmarc.com/blog/how-to-set-up-tls-rpt/
  - Generate DNS record: https://easydmarc.com/tools/tls-rpt-generator
  - Check DNS record: https://easydmarc.com/tools/tls-rpt-check
  - Generate MTA-STS DNS record: https://easydmarc.com/tools/mta-sts-generator
  - Check MTA-STS DNS record: https://easydmarc.com/tools/mta-sts-check
- MXToolbox info page: https://mxtoolbox.com/dmarc/smtp-tls/what-are-smtp-tls-reports
  - Check TLS-RPT DNS record: https://mxtoolbox.com/SuperTool.aspx (with tlsrpt: prefix)
  - Check MTA-STS DNS record: https://mxtoolbox.com/mta-sts.aspx
